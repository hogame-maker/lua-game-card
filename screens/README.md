# Screens

Esta pasta deve conter imagens de visualização (capturas de tela) do jogo.

Sugestões:
- `login.png` - tela de login/comeco
- `game.png` - tela principal do jogo
- `popup_saves.png` - modal de selecao de saves

Você pode gerar as capturas manualmente usando o LOVE (ou outra ferramenta) e salvar aqui.
